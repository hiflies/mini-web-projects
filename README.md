# Mini Web Projects

A collection of small web development projects built for learning, practice, and experimentation.  
This repository contains multiple independent mini projects created using HTML, CSS, and JavaScript.

Live site: [halime.tr](https://halime.tr)

---

## About

This repository is a personal learning space where I build small projects to improve my frontend development skills.  
Each folder contains a separate project with its own functionality and design.

The goal is to practice:
- HTML structure and semantics
- CSS styling and layout
- JavaScript logic and DOM manipulation
- Building real-world small UI components
- React (component-based architecture)
- Creative coding with p5.js

## Getting Started

Each project is standalone. Open any project's `index.html` directly in a browser, or serve it with a local dev server:

```bash
npx serve <project-folder>
```

The React project (`honeybee-cakes/`) requires Node.js — see its own folder for setup instructions.
